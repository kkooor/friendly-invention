## 一、物理层的基本概念

### 1. 物理层在体系结构中的位置

物理层是 OSI/RM 和 TCP/IP 协议栈的**最底层**，直接面向实际的物理传输介质。

### 2. 核心任务

- 确定与传输介质接口的**机械特性**（引脚数量、形状）、**电气特性**（电压范围）、**功能特性**（引脚定义）和**规程特性**（时序关系）。
- 为数据链路层提供**透明传输比特流**的服务。

### 3. 数据单元

物理层传输的数据单元是**比特（Bit）**。

------

## 二、数据通信基础

### 1. 通信系统模型

```
信源 → 发送器 → 传输系统 → 接收器 → 信宿
               ↑
         噪声源（干扰）
```

### 2. 基本术语

| 术语         | 定义                                                   |
| ------------ | ------------------------------------------------------ |
| **数据**     | 运送信息的实体。                                       |
| **信号**     | 数据的电气或电磁表现。                                 |
| **模拟信号** | 连续变化的信号（如声音波形）。                         |
| **数字信号** | 离散取值的信号（如 0/1 脉冲）。                        |
| **码元**     | 承载信息量的基本信号单元，一个码元可携带若干比特信息。 |

### 3. 通信方式分类

| 分类标准       | 类型         | 说明                           |
| -------------- | ------------ | ------------------------------ |
| 按信号传输方向 | **单工**     | 只能单向传输（广播）。         |
| 按信号传输方向 | **半双工**   | 可双向，但不能同时（对讲机）。 |
| 按信号传输方向 | **全双工**   | 可同时双向传输（电话）。       |
| 按信号同步方式 | **异步传输** | 以字符为单位，起止位同步。     |
| 按信号同步方式 | **同步传输** | 以数据块为单位，时钟同步。     |

## 三、信道的极限容量（核心定理）

物理层研究的核心问题是：**给定信道，最大能传输多快的数据？**

### 1. 奈奎斯特定理（Nyquist，理想无噪声信道）

给出了**带宽受限、无噪声**条件下的最大码元传输速率。

> **公式**：
>
> 最大码元传输速率：
>
> Bmax=2W(Baud)
>
> 数据传输速率：
>
> C=2W⋅log2V(bit/s)

- W：信道带宽（Hz）
- V：信号可取的离散电平数（码元种类数）
- Bmax：最大码元传输速率（波特率）

### 2. 香农定理（Shannon，有噪声信道）

给出了**带宽受限、有高斯噪声干扰**条件下的最大信息传输速率。

> **公式**：
>
> C=W⋅log2(1+NS)(bit/s)

- S/N：信噪比（常用分贝 dB 表示：10log10(S/N)）
- C：信道极限传输速率（**上界**，无论采用何种编码调制都无法突破）

### 📝 二者关系总结

| 定理         | 适用条件             | 决定因素                |
| ------------ | -------------------- | ----------------------- |
| **奈奎斯特** | 无噪声、理想低通信道 | 带宽 W 和信号离散等级 V |
| **香农**     | 有高斯噪声信道       | 带宽 W 和信噪比 S/N     |

> **实际传输速率 \**≤\** 奈奎斯特上限 且 \**≤\** 香农上限。**

## 四、编码与调制

将数字数据（比特）转换为物理信号的过程。

| 类型                    | 定义               | 典型方法                                                     |
| ----------------------- | ------------------ | ------------------------------------------------------------ |
| **数字数据 → 数字信号** | 编码               | 不归零编码 (NRZ)、曼彻斯特编码、差分曼彻斯特编码             |
| **数字数据 → 模拟信号** | 调制               | 幅移键控 (ASK)、频移键控 (FSK)、相移键控 (PSK)、正交振幅调制 (QAM) |
| **模拟数据 → 数字信号** | 脉冲编码调制 (PCM) | 采样 → 量化 → 编码                                           |
| **模拟数据 → 模拟信号** | 调制               | 调幅 (AM)、调频 (FM)、调相 (PM)                              |

### 重点：常用数字编码方案对比

| 编码方式          | 特点                                       | 优点           | 缺点                     |
| ----------------- | ------------------------------------------ | -------------- | ------------------------ |
| **NRZ（不归零）** | 正电平为 1，负电平为 0                     | 简单           | 无自同步能力，含直流分量 |
| **曼彻斯特编码**  | 位中心跳变：上跳为 0，下跳为 1             | 自带时钟同步   | 占用带宽加倍             |
| **差分曼彻斯特**  | 位中心总跳变，位开始有跳变为 0，无跳变为 1 | 抗干扰、易同步 | 实现复杂                 |

## 五、传输介质

传输介质分为**导向性**（有线）和**非导向性**（无线）两大类。

### 1. 导向性传输介质

| 介质         | 组成                      | 特点与应用                                                   |
| ------------ | ------------------------- | ------------------------------------------------------------ |
| **双绞线**   | 两根绝缘铜线绞合          | 成本低、易安装；分 STP（屏蔽）和 UTP（非屏蔽），广泛用于局域网。 |
| **同轴电缆** | 内导体 + 绝缘层 + 屏蔽层  | 抗干扰强、带宽高；用于有线电视、早期以太网。                 |
| **光纤**     | 纤芯 + 包层（全反射原理） | 带宽极高、损耗极低、抗电磁干扰。分多模光纤（短距）和单模光纤（长距）。 |

### 2. 非导向性传输介质（无线）

- **无线电波**：低频段，穿透性强，广播、Wi-Fi。
- **微波**：高频段，视距传输，地面微波接力、卫星通信。
- **红外线**：短距离、点对点，遥控器。
- **激光**：点对点大气激光通信。

## 六、信道复用技术

在一条物理链路上同时传输多路信号的技术。

| 复用技术         | 缩写 | 原理                                 | 典型应用           |
| ---------------- | ---- | ------------------------------------ | ------------------ |
| **频分复用**     | FDM  | 不同信号占用不同频段（频率分割）     | 广播电台、模拟电视 |
| **时分复用**     | TDM  | 不同信号占用不同时间片（时间分割）   | T1/E1 数字线路     |
| **统计时分复用** | STDM | 按需动态分配时间片，提高利用率       | 分组交换网         |
| **波分复用**     | WDM  | 光的频分复用，不同波长复用在一根光纤 | 光纤骨干网         |
| **码分复用**     | CDM  | 各用户使用不同正交码型，同时同频传输 | 3G 移动通信、GPS   |

### 重点：CDMA（码分多址）

- 每个站分配唯一的 **m 位码片序列**。
- 发送 1 时发送原码片，发送 0 时发送码片反码。
- 接收端用相同码片序列做**内积**运算恢复数据（归一化内积为 1 表示 1，-1 表示 0）。
- **优点**：抗干扰强、保密性好、频谱利用率高。

## 七、宽带接入技术（物理层视角）

| 技术                     | 原理                                     | 特点                                         |
| ------------------------ | ---------------------------------------- | -------------------------------------------- |
| **ADSL**                 | 利用电话线高频段传输数据，低频段保留语音 | 非对称速率（下行 > 上行），基于 FDM/DMT 调制 |
| **光纤同轴混合网 (HFC)** | 光纤到小区，同轴电缆入户                 | 基于 FDM，需 Cable Modem                     |
| **光纤接入 (FTTx)**      | 光纤直接延伸至用户附近或家中             | 高带宽、低损耗，PON（无源光网络）是主流      |

## 八、物理层设备

| 设备                  | 功能                                   | 备注                                         |
| --------------------- | -------------------------------------- | -------------------------------------------- |
| **中继器 (Repeater)** | 将衰减的信号**再生放大**，延长传输距离 | 工作在物理层，不识别帧结构                   |
| **集线器 (Hub)**      | 多端口的中继器，构成星型拓扑           | 共享带宽，同一时间只允许一个端口发送，易冲突 |

> **注意**：中继器和集线器都只负责**比特流的转发**，不进行任何差错校验或地址识别，因此无法隔离冲突域。

## 九、本章核心公式速查表

| 场景                   | 公式                   | 变量说明                    |
| ---------------------- | ---------------------- | --------------------------- |
| **无噪声信道最大速率** | C=2Wlog2V              | W：带宽 (Hz)，V：码元种类数 |
| **有噪声信道最大速率** | C=Wlog2(1+S/N)         | S/N：信噪比（比值，非 dB）  |
| **信噪比 dB 换算**     | dB=10log10(S/N)        | 例：30dB 对应 S/N=1000      |
| **波特率与比特率关系** | 比特率 = 波特率 ×log2V | V：一个码元携带的比特数     |
